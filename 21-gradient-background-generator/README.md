## Resumo do exercício

Este exercício propõe a criação de um gerador de gradiente CSS interativo utilizando HTML, CSS e JavaScript. O objetivo é permitir ao utilizador selecionar cores e direções para gerar um código CSS de gradiente, além de possibilitar a cópia fácil para a área de transferência.

### Funcionalidades Principais:

1.  **Seleção de Cores:**

- Utiliza dois inputs do tipo color para escolher as cores do gradiente.

2.  **Direção do Gradiente:**

- Botões interativos permitem escolher a direção do gradiente (para cima, para baixo, direita, esquerda, etc.).

3.  **Atualização Visual Instantânea:**

- A medida que o utilizador seleciona as opções, a visualização do gradiente é atualizada instantaneamente na página.

4.  **Geração de Código CSS:**

- Ao clicar no botão de submissão, o código CSS correspondente ao gradiente selecionado é gerado e exibido em um campo de texto.

5.  **Cópia para a Área de Transferência:**

- Um botão permite a cópia fácil do código CSS gerado para a área de transferência do utilizador.
