## Resumo do exercício

A aplicação é uma interface simples para contar números. Ela permite incrementar, decrementar e reinicializar o contador. Além disso, podemos definir um valor máximo para o contador e um número de começo.

**Funcionalidades Principais:**

1.  **Incremento e Decremento:**

- Botões para incrementar e decrementar o valor do contador.
- O valor do contador é exibido no display.

2.  **Reinicialização:**

- Botão para reinicializar o contador para zero.
- Também reinicializa o valor máximo, se definido.

3.  **Valor Máximo:**

- O utilizador pode definir um valor máximo para o contador.
- O botão de incremento é desativado quando o valor máximo é atingido.

4.  **Entrada de Valores:**

- Entradas para definir o valor inicial e máximo do contador.

5.  **Eventos:**

- Evento de carregamento inicial.
- Evento de alteração para as entradas de valores.
