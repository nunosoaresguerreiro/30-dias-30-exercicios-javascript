## Resumo do exercício

Este exercício apresenta uma aplicação visual interativa que exibe números de 1 a 100 em um grid. Cada número é classificado e estilizado com base nas seguintes categorias: números primos, números pares e números ímpares. O objetivo é ensinar conceitos básicos de manipulação do DOM, criação de elementos dinâmicos, e uso de lógica condicional em JavaScript.

**Funcionalidades Principais:**

1.  **Classificação de Números:**

- Primos: Números que só podem ser divididos por 1 e por eles mesmos.
- Pares: Números divisíveis por 2.
- Ímpares: Números que não são divisíveis por 2.

2.  **Geração Dinâmica do Grid:**

- Os números de 1 a 100 são gerados dinamicamente e exibidos como quadrados (divs) numa grid.
- Cada quadrado é estilizado com uma cor específica para identificar se o número é primo, par ou ímpar.

3.  **Estilização Visual:**

- Primos: Destacados com uma cor única.
- Pares: Estilizados com outra cor.
- Ímpares: Estilizados com uma terceira cor.

4.  **Código Modular e Reutilizável:**

- Uso de funções como isPrime para verificar se um número é primo.
- Estrutura clara e comentada para facilitar a compreensão e manutenção.
