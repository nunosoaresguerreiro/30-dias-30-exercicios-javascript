## Resumo do exercício

Este exercício apresenta uma aplicação simples de lista de tarefas (To-Do List). O objetivo é demonstrar conceitos básicos de manipulação do DOM, eventos, arrays e lógica em JavaScript. A aplicação permite adicionar, marcar como concluídas e remover tarefas, além de armazenar as informações em um array para futuras manipulações.

**Funcionalidades Principais:**

1.  **Adicionar Tarefas:**

- Input para digitar uma nova tarefa.
- Botão ou tecla "Enter" para adicionar a tarefa à lista.
- Cada tarefa é adicionada ao DOM e ao array de tarefas com informações sobre seu texto e estado inicial.

2.  **Marcar Tarefas como Concluídas:**

- Botão dedicado para marcar uma tarefa como concluída.
- A tarefa concluída muda de estilo visual (como sublinhado ou texto riscado).
- O estado da tarefa é atualizado no array.

3.  **Remover Tarefas:**

- Botão dedicado para remover uma tarefa da lista.
- A tarefa é removida do DOM e do array.
