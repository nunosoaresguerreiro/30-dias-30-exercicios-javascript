## Calculadora Simples em JavaScript

Este é um aplicativo de calculadora simples desenvolvido em JavaScript, que permite realizar operações matemáticas básicas diretamente no navegador.

### Funcionalidades

- Exibição Dinâmica dos Valores

- Os números e operadores são exibidos dinamicamente na tela conforme são pressionados.

- Suporta Operações Matemáticas Básicas

- Implementação de uma função safeEval() que valida a expressão antes da execução para evitar falhas de segurança.

- Tratamento de Erros

- Caso uma expressão inválida seja inserida, o dispay exibe "Error" em vez de executar comandos inseguros.
