## Resumo do exercício

1.  **Contagens Dinâmicas:**

- `char`: Mostra o número de caracteres no texto, excluindo espaços.
- `words`: Exibe a quantidade de palavras presentes no texto.
- `space`: Revela o número de espaços em branco no texto.

2.  **Exploração de Caracteres:**

- `first`: Apresenta o primeiro caractere do texto.
- `last`: Destaca o último caractere do texto, ignorando espaços no final.

3.  **Transformações:**

- `lower`: Transforma todo o texto em letras minúsculas.
- `upper`: Converte todo o texto para letras maiúsculas.

4.  **Capitalização Sutil:**

- `capitalized`: Capitalizar a primeira letra de cada palavra

5.  **Reset:**

- `resetBtn`: Ao tocar neste botão, restauramos tudo ao estado inicial.
