## Password Strength Checker 

Este é um simples script JavaScript para verificar a força de uma senha com base em vários critérios, incluindo a presença de letras maiúsculas, minúsculas, números e caracteres especiais, fornecendo feedback visual sobre a força da senha. 

### Critérios Verificados 

O script verifica os seguintes critérios: 

-  **Letras Maiúsculas:** Pelo menos uma letra maiúscula. 
-  **Letras Minúsculas:** Pelo menos uma letra minúscula. 
-  **Números:** Pelo menos um número. 
-  **Caracteres Especiais:** Pelo menos um caractere especial (por exemplo, !, @, #). 
-  **Comprimento Mínimo:** A senha deve ter pelo menos 8 caracteres.

### Funcionalidades

-   **Validação de Entrada:** Verifica se a senha está vazia ou excede 12 caracteres.
-   **Verificação de Força:** Avalia a força da senha com base nos critérios mencionados.
-   **Atualização do Interface:** Adiciona elementos de barra de força visual e mensagens com base na força da senha.
