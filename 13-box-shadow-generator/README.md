## Box Shadow Generator

Esta é uma ferramenta web simples que permite gerar interativamente código CSS para um efeito de sombra em caixas. A ferramenta fornece sliders para ajustar os deslocamentos X e Y, raio de desfoque e raio de propagação da sombra. Além disso, é possível escolher uma cor para a sombra. O código CSS gerado é exibido dinamicamente e pode ser facilmente copiado para a área de transferência.

### Sliders:
Mover os sliders para ajustar os deslocamentos X e Y, raio de desfoque e raio de propagação da sombra.

### Color Picker:
Clique na caixa de seleção de cor para abrir um seletor de cores e escolher a cor desejada para a sombra.

### Generated CSS Code:
A app exibirá dinamicamente o código CSS gerado com base nos seus ajustes.

### Copy to Clipboard:
Usar o botão "Copiar" para copiar o código CSS gerado para a área de transferência.
