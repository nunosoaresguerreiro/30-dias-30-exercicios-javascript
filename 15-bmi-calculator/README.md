## Resumo do exercício

Projeto simples de uma Calculadora de Índice de Massa Corporal (IMC) desenvolvida em JavaScript. O IMC é uma medida comum utilizada para avaliar se uma pessoa está dentro do peso considerado saudável para sua altura.

### Funcionalidades:

1.  **Cálculo do IMC:**

- O utilizador insere os dados na interface da aplicação.
- Ao submeter o formulário, o script realiza o cálculo do IMC.

2.  **Classificação do IMC:**

- O IMC é então classificado em categorias como "Abaixo do Peso", "Saudável", "Sobrepeso" ou "Obesidade".

3.  **Atualização Dinâmica:**

- O resultado é exibido dinamicamente na interface.
- As classificações do IMC e o valor calculado são apresentados ao utilizador.

4.  **Validação de Entrada:**

- A aplicação verifica se os valores inseridos são válidos (números positivos) antes de realizar os cálculos.

5.  **Reset:**

- Um botão de reset para reiniciar o formulário.

### Utilização:

1.  **Preenchimento do Formulário:**

- Insira seu peso e altura no formulário e clique em "Submit".

2.  **Resultados:**

- O resultado do cálculo do IMC será exibido, indicando a categoria do IMC e seu valor numérico.

3.  **Reset:**

- Use o botão "Reset" para limpar os campos e recomeçar.

### Classificação do IMC:

- **Abaixo de 18.5:** Abaixo do Peso
- **18.5 a 24.9:** Peso Saudável
- **25 a 29.9:** Sobrepeso
- **30 ou mais:** Obesidade
