## Conversor de Temperatura

Este é um simples conversor de temperatura construído que permite converter valores de temperatura entre Celsius, Fahrenheit e Kelvin.

### Instruções de Uso

1.  **Entrada de Dados**: Inserir o valor de temperatura que deseja converter no campo fornecido.
    
2.  **Seleção de Conversão**: Escolher a unidade de temperatura de origem (por exemplo, Celsius) e a unidade para a qual deseja-se converter (por exemplo, Fahrenheit ou Kelvin) marcando as caixas de seleção correspondentes.
    
3.  **Conversão**: Clique no botão "Converter" para calcular a conversão da temperatura.
    
4.  **Limpar Campos**: Se desejar limpar os campos de entrada e saída, utilizamos o botão de "Reset".
    

### Funcionalidades

-   **Validação de Entrada**: Verifica se o valor inserido é um número válido.

-   **Conversão de Temperatura**: Usa um mapa de conversão para calcular a temperatura convertida com base nas unidades selecionadas.

-   **Limpar Campos**: Permite limpar os campos de entrada e saída com o botão "Reset".
