## Resumo do exercício

Este exercício apresenta uma aplicação simples para gerir o estado de uma lâmpada. O objetivo é mostrar conceitos básicos de manipulação do DOM, eventos e lógica condicional em JavaScript. A lâmpada pode ser ligada, desligada, partida e substituída, dependendo das interações do utilizador.

**Funcionalidades Principais:**

1.  **Ligar e Desligar a Lâmpada:**

- Botão para ligar a lâmpada.
- Botão para desligar a lâmpada.
- A lâmpada só pode ser ligada ou desligada se não estiver partida.

2.  **Quebrar a Lâmpada:**

- Duplo clique na lâmpada para parti-la.
- Quando partida, a lâmpada não pode mais ser ligada ou desligada até ser substituída.

3.  **Substituir a Lâmpada:**

- Botão para substituir a lâmpada.
- A substituição redefine a lâmpada para o estado desligado.

4.  **Mensagens de Log:**

- Mensagens na consola informam o estado atual e as ações realizadas, como "Lamp is already on" ou "Lamp is broken."
