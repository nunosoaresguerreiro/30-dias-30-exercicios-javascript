## Resumo do exercício

Este exercício apresenta uma aplicação simples de geração de tabuadas de multiplicação. O objetivo é ensinar conceitos básicos de manipulação do DOM, validação de entradas, loops e organização visual com grids. A aplicação permite inserir um número de 1 a 10 e visualize a tabuada correspondente de forma dinâmica.

**Funcionalidades Principais:**

1.  **Gerar Tabuada:**

- Input para o utilizador inserir um número de 1 a 10.
- Botão para gerar a tabuada correspondente.

2.  **Validação de Entrada:**

- Verificação para garantir que o número inserido esteja dentro do intervalo permitido (1 a 10).
- Mensagem de erro exibida caso o input esteja fora do intervalo.

3.  **Exibição Dinâmica:**

- Cada cálculo da tabuada é exibido em formato de grid, com colunas para o número, operador, multiplicador, igual e resultado.
- O layout é limpo e atualizado a cada nova interação, garantindo que apenas a tabuada atual seja exibida.
